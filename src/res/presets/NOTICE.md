# Shortcut Presets — Notice

These files are **seed catalogs**, not the active runtime mapping. The active
mapping is the user's `shortcuts.json` in the config directory, which is created
from the selected preset on first run and edited independently thereafter.

Preset names refer to third-party applications for keyboard-shortcut
compatibility only:

- **Gwenview** is a trademark of the KDE community.
- **XnView MP** is a trademark of Pierre-e Gougelet / XnSoft.
- **IrfanView** is a trademark of Irfan Skiljan.

None of these projects endorse or are affiliated with thumbgrid. The mappings
here are **independent reconstructions** from each application's documented or
observed default shortcuts — they are not derived from the applications' source
code or configuration files.

`qimgv` is not a third-party reconstruction: it is the default scheme inherited
from thumbgrid's own upstream project (thumbgrid is a qimgv fork), which is why
it keeps its plain name instead of a "-style" qualifier.
